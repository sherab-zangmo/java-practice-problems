# Java Practice Problems

This repository contains my Java practice problems for learning OOP, classes, methods, ArrayList, and menu-driven console programs.

## Projects

### 1\. Student Grade Manager

A simple Java console program to manage student grades.

### 2\. Library System

A Java console program to add books, show books, borrow books, and return books.

### 3\. Contact System

A Java console program to manage contacts, including adding, viewing, and searching contact details.

## Topics Practised

* Classes and objects
* Constructors
* Encapsulation
* ArrayList
* Methods
* Scanner
* Menu loops
* Basic OOP design
